# Expense Tracker
A console-based Expense Tracker application developed using Python.
## Features
* Add Expense
* View Expenses
* Show Total Expense
* Delete Expense
* Category-wise Summary
* Highest Expense Analysis
* CSV-based Data Storage
* Error Handling
## Technologies Used
* Python
* CSV File Handling
