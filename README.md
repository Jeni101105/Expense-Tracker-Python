# Expense Tracker

A console-based Expense Tracker application developed using Python.

## Features

* Add Expense
* View Expenses
* Show Total Expense
* Delete Expense
* Category-wise Summary
* Highest Expense Analysis
* CSV-based Data Storage
* Error Handling

## Technologies Used

* Python
* CSV File Handling

## Project Structure

Expense_Tracker/

├── main.py

├── expenses.csv

└── README.md

## How to Run

1. Download the project
2. Open terminal in the project folder
3. Run:

python main.py
